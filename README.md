📦 Amazon Clone (HTML, CSS, JavaScript)

A front-end clone of Amazon’s e-commerce platform, inspired by SuperSimpleDev’s tutorial. This project replicates the look and feel of Amazon’s homepage and product listings using only HTML, CSS, and vanilla JavaScript.

🔑 Features
- Homepage Layout: Navigation bar, search bar, product categories, and footer.
- Responsive Design: Built with CSS flexbox and grid for mobile and desktop compatibility.
- Product Cards: Reusable components displaying product images, names, prices, and ratings.
- Interactive Elements:
  - Add-to-cart button with basic cart simulation
  - Dropdown menus and hover effects
  - Image sliders for product showcases

🎯 Learning Outcomes
- Strengthened front-end fundamentals (HTML structure, CSS styling, JS interactivity).
- Practiced DOM manipulation and event handling.
- Gained experience in building real-world style UI from scratch.
